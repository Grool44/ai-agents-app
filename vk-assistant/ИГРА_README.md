# 🎮 VK Базука - Мини-игра

Веб-игра с улучшениями и экономикой.

## Запуск

### 1. Установи Flask

```powershell
cd vk-assistant
py -m pip install flask
```

### 2. Запусти сервер

```powershell
py game_server.py
```

### 3. Открой в браузере

```
http://localhost:5000
```

---

## Как опубликовать в интернете

### Вариант 1: PythonAnywhere (Бесплатно)

1. Зайди на **https://www.pythonanywhere.com/**
2. Зарегистрируйся (бесплатный аккаунт)
3. Загрузи файлы игры:
   - `game_server.py`
   - `templates/index.html`
   - `game_data.json`
4. В настройках Web App укажи:
   - WSGI file: путь к `game_server.py`
5. Твой сайт будет доступен по адресу:
   ```
   https://твое_имя.pythonanywhere.com
   ```

### Вариант 2: Render (Бесплатно)

1. Зайди на **https://render.com/**
2. Создай GitHub репозиторий с файлами игры
3. Подключи репозиторий в Render
4. Выбери "Web Service"
5. Команда запуска: `python game_server.py`
6. Получишь домен: `https://твое-имя.onrender.com`

### Вариант 3: Vercel / Netlify

Подойдут для статических сайтов, но для Flask нужен бэкенд.

## Продвижение в Яндекс и VK

### Яндекс

1. **Яндекс.Вебмастер** (https://webmaster.yandex.ru/)
   - Добавь свой сайт
   - Проверь индексацию
   - Улучши мета-теги (title, description)

2. **SEO оптимизация**:
   - Добавь в `index.html`:
   ```html
   <meta name="description" content="VK Базука - интересная мини-игра">
   <meta name="keywords" content="игра, клик, база, улучшение">
   ```

### ВКонтакте

1. **Создай группу/паблик**
   - Название: "VK Базука - игра"
   - Добавь ссылку на игру

2. **Реклама**:
   - Посты в тематических группах
   - Таргетированная реклама (от 500 руб)
   - Конкурсы и розыгрыши

3. **VK Mini Apps** (для продвинутых)
   - Зарегистрируй приложение в https://dev.vk.com/
   - Интегрируй игру как мини-приложение VK
