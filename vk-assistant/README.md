# 🤖 VK Assistant (Личный бот)

Python-ассистент для ВКонтакте: группы, сообщения, комментарии, сравнения.

## ⚡ Быстрый старт

### 1. Установка зависимостей

```bash
cd vk-assistant
pip install -r requirements.txt
```

### 2. Получение токена (ОБЯЗАТЕЛЬНО!)

Для **личного бота** нужен **User Access Token**, а не сервисный ключ!

1. Перейди на [vkhost.github.io](https://vkhost.github.io/)
2. Выбери приложение **VK Admin** или **Kate Mobile**
3. Нажми **Получить**, скопируй токен из адресной строки
4. Вставь его в файл `.env`

> ⚠️ Сервисный ключ НЕ подходит для отправки сообщений и чтения диалогов!

### 3. Настройка

```bash
# Скопируй шаблон
cp .env.example .env

# Открой .env и вставь токен
```

Пример `.env`:
```env
VK_USER_TOKEN=vk1.a.xxx...
VK_APP_ID=51877311
```

### 4. Запуск

```bash
python main.py
```

## 📁 Структура

| Файл | Назначение |
|------|-----------|
| `config.py` | Читает `.env` |
| `vk_client.py` | Основной класс VkAssistant |
| `main.py` | Примеры вызовов |
| `.env` | Секреты (не попадает в git) |

## 🛠 Возможности

- `get_my_groups()` — мои группы с количеством участников
- `search_groups(q)` — поиск групп
- `get_group_info(id)` — детали группы
- `compare_groups([id1, id2])` — сравнение групп
- `get_conversations()` — список диалогов
- `get_history(user_id)` — переписка с человеком
- `send_message(user_id, text)` — отправка сообщения
- `get_comments(owner_id, post_id)` — чтение комментариев
- `create_comment(...)` — написать комментарий
- `compare_users([id1, id2])` — сравнение профилей

## ⚠️ Лимиты VK API

- ~3 запроса в секунду для user token
- Для `messages.send` нужен токен с правом `messages`
- Некоторые методы недоступны без специального доступа

## 🔒 Безопасность

- **Никогда** не коммить файл `.env`
- Если токен утёк — отзови его в настройках приложения VK
